Tiptap Image Plus

An advanced image extension for Tiptap that adds extra options like resize, align, caption, and custom styles.
It makes handling images in your editor more powerful and user-friendly.

✨ Features

Section titled “✨ Features”
  • 🖼️ Resize images with drag handles
  • ↔️ Align images (left, center, right)
  • 🎨 Custom wrapper and container styles
  • 📱 Mobile-friendly controls
  • ⚡ Easy integration with Tiptap v2

Note: tiptap-image-plus extends the default Tiptap image extension with additional functionality.
